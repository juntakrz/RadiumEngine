DEVELOPMENT TOOLS
=================


1. "compileShaders_Win_x64_DEBUG"
Runs automatically when using a debug build of Radium Engine, compiles all GLSL shaders into SPIR-V format with the debug flag

2. "glTF_PNGtoKTX2"
Converts glTF-standard PNG files to an appropriately compressed KTX2 format. NVidia Texture Tools must be present in system PATH
